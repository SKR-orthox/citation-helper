# Multi-Site Citation Helper

🌐 사용 가능한 언어:  
[English](README.md) | [한국어](README.ko.md) | [日本語](README.ja.md)

PubMed, Google Scholar, Nature, ScienceDirect, Cell Press, The Lancet, SpringerLink,  
Wiley Online Library, Web of Science, Scopus 등 주요 학술 사이트에서  
논문 정보를 자동으로 추출하여 참고문헌을 생성해주는 Firefox 확장 프로그램입니다.

Vancouver 스타일과 보고서용 형식을 지원하며,  
UI 언어는 **영어 / 한국어 / 일본어**를 제공합니다.

---

## 주요 기능

- 다양한 학술 사이트에서 논문 정보 자동 추출  
- Vancouver 스타일 및 보고서용 형식 지원  
- 영어 / 한국어 / 일본어 UI  
- 원클릭 클립보드 복사  
- 깔끔하고 간단한 팝업 인터페이스

---

## 지원 사이트

- PubMed  
- Google Scholar  
- Nature  
- ScienceDirect (Elsevier)  
- Cell Press  
- The Lancet  
- SpringerLink  
- Wiley Online Library  
- Web of Science (일부 제한)  
- Scopus (일부 제한)

*로그인이나 동적 렌더링이 강한 사이트에서는 자동 추출이 제한될 수 있습니다.*

---

## 사용 방법

1. Firefox에서 확장 기능을 설치합니다.  
2. 지원되는 사이트에서 논문 상세 페이지를 엽니다.  
3. 확장 아이콘을 클릭합니다.  
4. 인용 형식과 UI 언어를 선택합니다.  
5. **인용 불러오기** 버튼을 누릅니다.  
6. 생성된 참고문헌을 **복사하기** 버튼으로 복사합니다.

---

## 아이콘

아이콘은 이 프로젝트를 위해 AI 도구를 활용하여 직접 제작했으며,  
다음 크기의 투명 PNG로 제공됩니다.

- 16×16  
- 32×32  
- 128×128
