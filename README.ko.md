# Citation Helper (PubMed MVP)

🌐 언어:  
[English](README.md) | [한국어](README.ko.md) | [日本語](README.ja.md)

PubMed 논문 상세 페이지에서  
참고문헌 정보를 정확하게 추출하여  
표준 인용 형식으로 변환해주는 Firefox 확장 프로그램입니다.

이 프로젝트는 **지원 사이트 수보다 인용 형식의 정확성**에 중점을 둡니다.

---

## 주요 기능

- PubMed 논문 자동 인용 추출
- 지원 인용 형식
  - Vancouver
  - APA 7판
  - IEEE
  - BibTeX
- 저자 수 규칙 정확히 처리
  - Vancouver: 6명 초과 시 et al.
  - APA 7: 21명 이상 말줄임표 처리
- DOI / PMID 자동 포함
- UI 언어 지원
  - 영어 / 한국어 / 일본어
- 원클릭 복사
- 단순하고 깔끔한 UI

---

## 지원 사이트

- PubMed  
  https://pubmed.ncbi.nlm.nih.gov/

*MVP 버전에서는 PubMed만 공식 지원합니다.*

---

## 사용 방법

1. Firefox에 확장 프로그램을 설치합니다.
2. PubMed 논문 상세 페이지를 엽니다.
3. 확장 아이콘을 클릭합니다.
4. 인용 형식과 언어를 선택합니다.
5. **인용 불러오기** 버튼을 누릅니다.
6. **복사하기** 버튼으로 참고문헌을 복사합니다.
