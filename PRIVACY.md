# Privacy Policy

Citation Helper does not collect personal information and does not track users.

## Data stored locally

Citation Helper stores the following settings in your browserâ€™s local extension storage to keep your preferences between sessions:

- uiLanguage (UI language)
- citationStyle (citation format/style)
- authorPreset (author name preset)
- debugMode (debug logging on/off)
- settingsVersion (internal settings schema version for migration)

This data is stored only on your device and is not sent to any server.

## Network requests

Citation Helper does not send any data to external services.

- No external API requests are made.
- No analytics or tracking requests are made.
