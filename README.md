# Citation Helper

PubMed 및 주요 학술 저널 사이트에서 논문 정보를 자동으로 추출하여  
Vancouver 스타일 또는 보고서용 참고문헌 형식으로 변환해주는 Firefox 확장 프로그램입니다.

---

## 주요 기능

- PubMed, Nature, Springer, ScienceDirect, Cell, The Lancet, Wiley 등 주요 학술 사이트 지원
- 논문 제목, 저자, 저널명, 연도, 권·호, 페이지 자동 추출
- Vancouver 스타일 / 보고서 형식 즉시 변환
- 한국어 / 영어 / 일본어 UI 지원
- 원클릭 클립보드 복사 기능

---

## 지원 사이트

- PubMed
- Google Scholar
- Nature
- ScienceDirect (Elsevier)
- Cell Press
- The Lancet
- SpringerLink
- Wiley Online Library
- Web of Science (일부 제한)
- Scopus (일부 제한)

※ 로그인이나 강한 동적 로딩이 필요한 사이트는 자동 추출이 제한될 수 있습니다.

---

## 사용 방법

1. Firefox에서 확장 프로그램을 로드합니다.
2. 논문 상세 페이지로 이동합니다.
3. 확장 프로그램 아이콘을 클릭합니다.
4. 형식(Vancouver / 보고서)과 언어를 선택합니다.
5. 「인용 불러오기」 버튼을 누릅니다.
6. 생성된 참고문헌을 「복사하기」 버튼으로 복사합니다.

---

## 아이콘 및 디자인

아이콘은 특정 상표나 로고를 모방하지 않고,  
AI를 활용해 프로젝트 용도로 제작된 오리지널 디자인이며  
투명 배경 PNG(16×16 / 32×32 / 128×128) 크기로 제공됩니다.
