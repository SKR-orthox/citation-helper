# 0.7.0 Manual Test Cases

이 문서는 0.7.0 릴리즈의 수동 테스트 체크리스트입니다.
각 URL에서 확장프로그램을 실행하고 결과가 맞으면 체크합니다.

태그 규칙(예시)
- DOI_Y / DOI_N
- PAGES_Y / PAGES_N / ELOCATOR
- ISSUE_Y / ISSUE_N
- AUTHORS_1 / AUTHORS_6 / AUTHORS_7 / AUTHORS_20 / AUTHORS_21P
- CONSORTIUM_Y / CONSORTIUM_N

검증 항목(각 케이스 공통)
- 상태가 Success인지
- Vancouver 출력이 규칙대로인지
- APA7 출력이 규칙대로인지
- DOI/Pages/Issue 누락 시 문장이 깨지지 않는지
